Comment fonctionnent les alertes ?
Création automatique

Les alertes sont générées automatiquement lorsque le système détecte une qualité d'eau MODERATE ou BAD lors d'une prédiction ou observation.

Types de sévérité
  • CRITICAL : Score < 50 ou qualité BAD
  • WARNING : Qualité MODERATE
  • INFO : Information générale
Cycle de vie
  1. OPEN : Nouvelle alerte détectée
  2. ACKNOWLEDGED : L'alerte a été vue
  3. RESOLVED : Le problème est résolu
